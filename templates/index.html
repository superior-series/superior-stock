⚙️ API設定

Unsplash
Unsplashでキーを取得する ↗
キー取得の手順を見る
  1. 上のリンクから Unsplash の開発者ページへ
  2. 「New Application」をクリック
  3. 規約のチェックボックスをすべてチェック
  4. アプリ名（例: Superior）と説明（例: A personal tool to collect royalty-free image URLs.）を入力
  5. 「Create application」を押すと Access Key が発行される
  6. そのキーをコピーして、下の入力欄に貼り付け → 保存
保存済みキー
Pexels
Pexelsでキーを取得する ↗
キー取得の手順を見る
  1. 上のリンクから Pexels の API 申請ページへ
  2. ログイン（未登録ならサインアップ）
  3. アプリ名と用途を入力（例: Superior Stock / Personal tool for collecting royalty-free image URLs.）
  4. 送信すると API Key がすぐ表示される
  5. そのキーをコピーして、下の入力欄に貼り付け → 保存
保存済みキー
Pixabay
Pixabayでキーを取得する ↗
キー取得の手順を見る
  1. 上のリンクから Pixabay の API ページへ
  2. ページ内で「Log in」または「Sign up」でアカウントを作成／ログイン
  3. ログイン後、同じページに戻ると自分の API Key が表示されている
  4. その Key をコピーして、下の入力欄に貼り付け → 保存
