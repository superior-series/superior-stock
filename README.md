# Superior Stock

フリー画像URLコレクター。Unsplash / Pexels / Pixabay の3つの正規APIから、
商用利用可能な画像のURLを一括で集めて整理できるWebアプリ。

## 特徴

- 完全に権利がクリーン（CC0 / Unsplash License / Pexels License）
- 商用利用可能
- 3つのソースから同時検索
- Excel形式でダウンロード可能

## APIキーの取得

各サービスで無料のAPIキーを取得して、アプリ内の「API設定」から登録してください。

- [Unsplash](https://unsplash.com/oauth/applications)
- [Pexels](https://www.pexels.com/api/new/)
- [Pixabay](https://pixabay.com/api/docs/)
